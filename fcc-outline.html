CSS (1156 steps total)
  • Computer Basics (16 steps)

    Get a quick refresher on how computers work, including the fundamental concepts of file systems, text editors, and browsers. This section ensures you have a solid foundation before diving into coding.

  • Basic CSS (126 steps)

    Dive into the world of CSS! You'll learn what CSS is, how to add it to your HTML, and the basic syntax for writing CSS rules. We'll cover selectors, properties, and values.

  • Design (23 steps)

    Learn the basics of web design theory, including color palettes, typography, and visual hierarchy. You'll understand how to make your websites look good and be easy to read.

  • Absolute and Relative Units (8 steps)

    Explore different ways to size your elements. This section will teach you the difference between fixed units like pixels and responsive units like percentages and viewport units.

  • Pseudo Classes and Elements (37 steps)

    Learn how to target specific states of an element, like when a user hovers over a button, and how to style specific parts of an element, such as the first letter of a paragraph.

  • Colors (98 steps)

    Master the art of using color on the web. We'll cover different color models like RGB, HEX, and HSL, and how to apply them to create beautiful designs.

  • Styling Forms (67 steps)

    Transform plain HTML forms into visually appealing and user-friendly components. You'll learn how to style input fields, buttons, and other form elements.

  • The Box Model (54 steps)

    Understand the most important concept in CSS layout: the box model. You'll learn about content, padding, borders, and margins, and how they affect the size and placement of your elements.

  • Flexbox (27 steps)

    Learn to use CSS Flexbox, a powerful layout module that makes it easy to align and distribute space among items in a container, even when their size is unknown or dynamic.

  • Typography (78 steps)

    Deepen your understanding of web typography. This section covers font families, font sizes, line height, and other properties that make your text readable and aesthetically pleasing.

  • Accessibility (72 steps)

    Build on your HTML accessibility knowledge by learning how CSS can be used to improve the user experience for everyone, including managing color contrast and focus states.

  • Positioning (88 steps)

    Master CSS positioning, from static and relative to absolute and fixed. You'll learn how to place elements precisely where you want them on the page.

  • Attribute Selectors (72 steps)

    Learn advanced CSS selectors that allow you to style elements based on their attributes, such as `[type="text"]` or `[title]`. This provides more granular control over your styling.

  • Responsive Design (38 steps)

    Learn the principles of responsive web design. You'll use media queries to create layouts that adapt to different screen sizes and devices, ensuring your site looks great everywhere.

  • Variables (120 steps)

    Discover the power of CSS variables (custom properties). You'll learn how to define and use them to make your stylesheets more maintainable and reusable.

  • Grid (92 steps)

    Explore CSS Grid, a two-dimensional layout system that gives you complete control over rows and columns. It's the ideal tool for creating complex, grid-based layouts.

  • Animations (139 steps)

    Bring your web pages to life with CSS animations and transitions. You'll learn to create smooth, eye-catching effects and interactive animations without needing JavaScript.

  • **REVIEW: CSS Review**

    A final review of all major CSS concepts before the certification exam. We'll cover everything from the box model and Flexbox to Grid and animations.

JavaScript (917 steps total)
  • Code Editors (5 steps)

    Get familiar with the tools you'll use to write JavaScript. We'll cover basic features of a code editor that will help you be more productive.

  • Variables and Strings (62 steps)

    Start with the basics of JavaScript: how to declare variables, and how to work with text data using strings. You'll learn about concatenation, templates, and string methods.

  • Booleans and Numbers (37 steps)

    Learn about different data types, including numbers for mathematical operations and booleans for logic and decision-making in your code.

  • Functions (37 steps)

    Master the concept of functions, reusable blocks of code that perform a specific task. You'll learn how to define, call, and pass arguments to functions.

  • Arrays (34 steps)

    Explore arrays, a fundamental data structure for storing collections of data. You'll learn to access, modify, and loop through array elements.

  • Objects (28 steps)

    Discover objects, which are used to store data in key-value pairs. You'll learn how to create objects, access properties, and use object methods.

  • Loops (21 steps)

    Learn how to repeat a block of code using different types of loops, such as `for` and `while` loops, to perform repetitive tasks efficiently.

  • JavaScript Fundamentals Review

    A comprehensive review of the JavaScript fundamentals you've learned so far, including variables, data types, and basic control flow.

  • Higher Order Functions and Callbacks (29 steps)

    Learn about advanced function concepts like higher-order functions and callbacks, which are essential for asynchronous programming and modern JavaScript frameworks.

  • DOM Manipulation and Events (72 steps)

    Learn how JavaScript interacts with HTML and CSS. You'll manipulate the Document Object Model (DOM) to change content and style, and handle user events like clicks and key presses.

  • JavaScript and Accessibility (3 quiz pages)

    Understand how to use JavaScript to create accessible interactive components, focusing on proper use of ARIA and managing dynamic content for screen readers.

  • ES6 Features (58 steps)

    Get up to speed with modern JavaScript (ES6 and beyond). We'll cover topics like `const` and `let`, arrow functions, and destructuring.

  • Promises and Asynchronous JavaScript (42 steps)

    Master asynchronous operations in JavaScript using Promises, async/await, and the Fetch API to handle data from external sources without blocking the main thread.

  • Object-Oriented Programming (35 steps)

    Learn the core principles of OOP, including classes, objects, inheritance, and polymorphism. You'll learn how to structure your code in a more organized way.

  • Modules (18 steps)

    Learn about JavaScript modules, which allow you to organize your code into separate, reusable files. This makes your projects more maintainable and scalable.

  • Data Structures and Algorithms (50 steps)

    Explore common data structures like stacks, queues, and linked lists, and learn fundamental algorithms for searching and sorting data.

  • **REVIEW: JavaScript Review**

    A final review of all major JavaScript concepts, from fundamentals to advanced topics like asynchronous programming and OOP, to prepare for your final exam.

  • **QUIZ: JavaScript Quiz (20 questions)**

    Test your knowledge of JavaScript concepts, from basic syntax to more complex ideas, with a series of multiple-choice questions.

Testing (24 steps total)
  • Testing Intro (8 steps)

    Understand the importance of testing in software development. This section will introduce you to the different types of tests and the benefits of writing them.

  • Test Driven Development (9 steps)

    Learn the TDD methodology: writing tests before you write the code they are meant to test. This practice helps ensure your code is robust and free of bugs.

  • Automated Tests (7 steps)

    Get an introduction to writing automated tests using popular JavaScript testing frameworks. You'll learn how to set up a test environment and write your first test cases.

  • Unit Testing (15 steps)

    Focus on unit testing, which involves testing individual functions or components in isolation to ensure they work as expected.

  • Integration Testing (10 steps)

    Learn how to write integration tests to ensure that different parts of your application work together correctly as a group.

  • End-to-End Testing (12 steps)

    Understand end-to-end testing, a type of test that simulates a real user's journey through your application to ensure the entire system is working correctly.

  • QA and Bugs (25 stories)

    Learn about Quality Assurance (QA) and how to identify and debug common issues in a web application. This section is more about practice and problem-solving.

  • **REVIEW: Testing Review**

    A final review of all testing concepts, including unit, integration, and E2E testing, to prepare you for the final project and certification exam.
